# 07 — 元服务卡片"今日相关"

**What to build:** Form Kit 桌面元服务卡片,常驻展示"今天会用到的截图":今日 DDL、待取快递、当天课表,多条时自动轮换。卡片只读 RelationalStore 的"今日相关"视图,与主 App 解耦——做不完不影响主线,做完是演示视频里最直观的鸿蒙特色画面。

**Blocked by:** 02 — 核心曳光弹

**Status:** ready-for-agent

- [ ] 桌面可添加拾光卡片
- [ ] 库中存在今日事件时,卡片正确展示其摘要与缩略图
- [ ] 多个今日事件自动轮换展示
- [ ] 无今日事件时展示友好空态(而非空白)
- [ ] 日期变更后卡片内容随之更新
- [ ] 点击卡片跳转到 App 内对应卡片详情
