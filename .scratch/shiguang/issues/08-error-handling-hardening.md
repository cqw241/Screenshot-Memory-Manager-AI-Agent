# 08 — 健壮性硬化:错误分类分流

**What to build:** 把 spec 的错误处理表落地为解析管线的实际行为,使任何一层失败都降级不阻断:

| 错误类别 | 行为 |
|---|---|
| 认证失败(401/403) | 停止排队,通知栏提示"请检查 API Key",引导去设置页 |
| 运行期"不支持图片"(厂商改接口等意外) | 暂停解析队列,通知提示"当前模型不再支持图像,请更换模型" |
| 限流/超时(429/5xx) | 指数退避排队重试,不丢任务 |
| 无网络 | 摄取照常、解析排队,联网后自动续跑 |
| OCR 失败 | 存原图待重试 |
