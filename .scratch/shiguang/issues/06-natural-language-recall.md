# 06 — 自然语言召回对话

**What to build:** App 内对话页。用户用模糊问法提问("上个月截的那个健身房优惠还在吗""帮我找有二维码的那张"),系统先按结构化字段(类型/日期/实体)过滤候选,再把候选卡片摘要送入 LLM 做语义挑选,回答附原图,且带记忆 + 建议的语气("你 3 月 2 日截的,还有 5 天过期,要现在用吗?"),支持一键写入日程。不做向量库。

**Blocked by:** 02 — 核心曳光弹

**Status:** ready-for-agent

- [ ] "我上次截的那个奖学金申请截止是哪天?"→ 返回正确卡片、原图和截止日期
- [ ] 按时间范围模糊提问("上个月的")能正确过滤
- [ ] 无匹配时坦率回答"没找到",并给出可能原因(如"那段时间没有解析记录")
- [ ] 回答中的建议与卡片状态一致(过期的说过期,临期的给天数)
- [ ] 回答附带的「写入日程」动作生效
