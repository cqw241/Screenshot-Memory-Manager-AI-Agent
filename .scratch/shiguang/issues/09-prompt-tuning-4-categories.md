# 09 — Prompt 打磨:4 类截图识别达标

**What to build:** 建立一组固定测试截图集(讲座海报 / 取件码 / DDL 通知 / 优惠券,每类不少于 10 张真实样本,含边缘情况:模糊、长截图、深色模式、图文混排),迭代解析 prompt 与 JSON schema,直到识别质量达标。不认识的类型稳定落"未分类",而不是错分类。宁可 4 类丝滑,不要 20 类半残。

**Blocked by:** 03 — 静默摄取

**Status:** ready-for-agent

- [ ] 测试集 ≥40 张(4 类 × ≥10),覆盖深色模式/模糊/长截图等边缘情况
- [ ] 类型识别准确率 ≥90%,且错误集中在"落未分类"而非"错分类"
- [ ] 关键实体(时间/地点/码号/金额)在类型正确的样本中提取准确率 ≥95%,取件码/金额零字符错误(OCR 对照生效)
- [ ] 测试集与跑分脚本/清单入库,便于换模型后回归
- [ ] 非 4 类截图(风景照截图、表情包)稳定落"未分类",不产生错误提醒
