# 03 — 静默摄取:截图自动监听

**What to build:** 用户正常使用手机截图,无需打开拾光。App 通过 photoAccessHelper 监听媒体库新增截图,自动走 02 的解析管线;完成后通知栏轻提示一次"已读懂 1 张截图",点开可查看结果并纠错(修改类型)。交互克制:每张截图只提示一次,不打扰。

**Blocked by:** 02 — 核心曳光弹

**Status:** ready-for-agent

- [ ] 在任意界面正常截图,不打开 App,数秒后收到"已读懂 1 张截图"轻提示
- [ ] 点开提示可见解析结果,可手动修改类型并保存
- [ ] 打开 App,新卡片已出现在时间轴
- [ ] 仅新增截图触发解析,普通拍照/保存图片不触发
- [ ] 相册权限被拒时给出清晰引导,不崩溃
