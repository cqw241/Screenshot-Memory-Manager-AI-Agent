# 赛事主题：AIAgent创新
1. 作品描述
围绕个人日常生活与陪伴、主动服务、自然交互，开展AI智能体验的创新设计与开发。
作品创作不限应用场景，学习效率、学业管理、生活服务、内容创作、运动健康、消费购物、社交情感、智慧出行、求职招聘等各类生活与校园场景，均可自由发挥创作。

2. 作品解析
个人日常生活与陪伴：在日常生活、工作学习、成长陪伴等场景，让AI成为专属伙伴；
主动服务：结合位置、时间、拍照、对话历史等用户上下文信息，让AI主动出现在对的时刻；
自然交互：探索Agent在不同情景下的最佳交互形式，涵盖文本、语音、视觉及创新硬件结合；

## 作品要求

| 序号    | 提交项         | 要求                                                                                                                                           |
| ----- | ----------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **创意描述**    | 一句话描述关键创新点。                                                                                                                                                                               |

## 评审标准

**总分 = 基础分 + 附加分**

| 评分维度            |       分值 | 评分内容                                   |
| --------------- | -------: | -------------------------------------- |
| **创新性**         | **50 分** | 产品创意、鸿蒙技术创新性、交互创新、技术创新                 |
| **完备度**         | **20 分** | 体验完整顺畅、功能完备度                           |
| **前景评估**        | **20 分** | 需求真实性、社会价值、可落地商业价值                     |
| **规范性**         | **10 分** | 文档规范性、功能演示清晰、内容表达清晰度                   |
